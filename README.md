# 安装Nodejs

首先需要保证本地已经安装好 NodeJs开发环境



# 使用 vue-cli 创建项目的基本框架

前往 vue-cli 可点击此处：[vue-cli](https://cli.vuejs.org/zh/guide/creating-a-project.html#vue-create)

## 执行 create 命令

```shel
vue create vue-project-template
```

- Babel
- Router
- Vuex
- CSS - processors
- Linter / Formatter
